ease-avatar

CSS-only avatar component for user profile pictures, initials fallback, and status indicators.

Image Avatar

Wrap an <img> with .ease-avatar for automatic rounding and sizing.

.ease-avatar

Initials

Add .ease-avatar-initials for text-based fallback with brand background.

JD .ease-avatar-initials
AB
MK

Sizes

XS xs (24px) .ease-avatar-xs
SM sm (32px) .ease-avatar-sm
MD md (40px) .ease-avatar (default)
LG lg (56px) .ease-avatar-lg
XL xl (80px) .ease-avatar-xl

Shapes

R Rounded (circle) .ease-avatar-rounded
S Square .ease-avatar-square
S Soft .ease-avatar-soft

Status Dot

Add .ease-avatar-status with -online, -away, or -busy.

Usage

<!-- Image avatar -->
<div class="ease-avatar">
  <img src="photo.jpg" alt="User name" />
</div>

<!-- Initials fallback -->
<span class="ease-avatar ease-avatar-initials">JD</span>

<!-- Sizes: .ease-avatar-xs, -sm, -lg, -xl -->
<!-- Shapes: .ease-avatar-square, .ease-avatar-soft -->

<!-- Status dot -->
<span class="ease-avatar ease-avatar-initials
  ease-avatar-status ease-avatar-status-online">JD</span>

<!-- Group -->
<div class="ease-avatar-group">
  <span class="ease-avatar ease-avatar-initials ease-avatar-sm">JD</span>
  <span class="ease-avatar ease-avatar-initials ease-avatar-sm">AB</span>
</div>
