# Animated Glassmorphism Pricing Card

Premium glassmorphic pricing card with hover-triggered 3D tilt, glowing border highlights, and billing cycle toggle.

## Features

- Glassmorphism design with backdrop blur and semi-transparent backgrounds
- 3D perspective tilt on hover (CSS `perspective` + `rotateX`/`rotateY`)
- Glowing neon border highlight on hover
- Monthly / Annual billing toggle with animated switch
- Two plan tiers (Starter, Pro) with feature lists
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Two-card pricing section with toggle
- `style.css` — Self-contained CSS (220 lines)
