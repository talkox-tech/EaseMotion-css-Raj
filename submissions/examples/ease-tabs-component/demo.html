ease-tabs

CSS-only tabs with animated underline — Issue #260

Overview

A CSS-only tabs component using hidden radio inputs and the sibling combinator. No JavaScript needed. The animated underline uses a scaleX transform with a bounce easing.

Features
  • Pure CSS — zero JavaScript
  • Animated underline indicator with bounce easing
  • Fade-in panel transition
  • Accessible via keyboard (radio inputs)
  • Respects prefers-reduced-motion
Usage

Wrap in .ease-tabs, add radio inputs with matching id/for pairs, then use .ease-tabs-nav + .ease-tab-label for buttons and .ease-tabs-panels + .ease-tab-content for panels.
